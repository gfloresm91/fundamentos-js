- Funciones
- Objetos

- Parámetros como referencia o valor
- Comparaciones
- Condicionales
- Arrow functions
- Ciclos: for y switch
- Arrays: Filter, map y reduce
- Clases y herencia
- setTimeOut
- Callbacks
- Asincronismo: async, await
- Promesas
